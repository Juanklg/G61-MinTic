Parrafos

En este texto vamos a mostar las diferentes etiquetas que podemos utilizar dentro de los parrafos o de cualquier etiqueta de texto. la primera es Strong, se usa para indicar que un texto o parte de un párrafo es importante. la segunda es time se usa para indicar que el texto hace referencia a una fecha 16/09/2021

En este parrafo vemos etiquetas para el texto que no relsatan informacion importante pero si modifican la forma en la que se ve el texto en la pagina, mostraremos algunos.

La etiqueta b : coloca el texto en negrita.
La etiqueta u : subraya el texto.
El br que hace un salto de linea
La etiqueta i : es una etiqueta coloca el texto en cursiva
La etiqueta q : representa una cita textual entre comillas
La etiqueta a : que genera un link a otra pagina u otro recurso con un nombre especifico youtube
